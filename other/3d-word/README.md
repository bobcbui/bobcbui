# 2D 开放世界游戏

这是一个基于 Phaser 3 的 2D 修仙（仙侠）风格开放世界原型，当前实现了自动范围攻击、拾取、以及基础的成长系统框架。

快速运行

- 在项目目录启动静态服务器并在浏览器打开：

```bash
python -m http.server 8000
```

默认地址： http://localhost:8000

控制与说明

- 鼠标左键：按住移动到光标位置（点击或拖动）
- 鼠标右键：在地图上放置标记
- Q / W / E / R：技能（示例：盘旋飞剑、玄雷术、气劲斩、御剑术）
- 1-4：消耗品（丹药 / 增益）
- G / I：打开或关闭背包
- O / V：设置或切换视角（占位）

当前特性

- 自动攻击：玩家会定期对一定半径内所有敌人造成伤害并显示灵力光环特效
- 敌人、掉落：敌人死亡可掉落丹药，拾取后增加消耗品
- 进阶系统：经验 / 等级 / 金币 / 技能书（雏形）
- UI：保留血量、状态与热栏等面板

开发说明

- 主要实现：[index.html](index.html)（已将 Babylon 替换为 Phaser，并在其中实现 2D 场景与逻辑）
- 如需调整数值（伤害、攻击半径、刷新率等），请编辑 `index.html` 中的 `playerState`、`progression` 等变量
- 后续建议：添加保存/载入、技能图标、AI 行为、更多关卡与更完善的 UI
