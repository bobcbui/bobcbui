# 认识Java

Java是一个电脑程序，这个程序有Windows版本、Mac版本，还有Linux版本。只要你的电脑可以安装Java那么就可以运行你用Java编写的程序，所以Java是一种跨平台的编程语言。

## Java可以做什么？

Java和其他程序不一样，我们可以用这个程序来开发另一个程序。它是一个没有图形界面的程序，你需要给它一段代码，它会根据你编写的内容来运行，显示出你想要的结果。

你可以给Java一段代码，让它在桌面上显示一个消息框，这段代码我们通常称之为`Java开发的程序`。

**比如：**
``` java
import javax.swing.JOptionPane;

public class HelloWorld {
    public static void main(String[] args) {
        JOptionPane.showMessageDialog(null, "HelloWorld");
    }
}
```
上面这个`Java程序`只是简单地弹出一个对话框，我们可以通过更复杂的代码来实现更丰富的功能。

比如：
- 模仿实现QQ、微信等聊天软件
- 实现一个记事本
- 定时切换电脑背景图
- 自动操作电脑
- 自动操作浏览器、网站
- 开发一个游戏
- 开发一个网站

当然还可以做更多。

## 小结

现在我们知道Java是一个电脑程序，运行它的方式就是给它一段代码，它就可以根据你编写的内容来运行，显示出你想要的结果。接下来的一段时间里，我们就来学习如何编写这段代码，也就是大家常说的编程语言。

## 名词介绍
- 程序： 这里指的程序是计算机程序，代码打包后的可执行文件都可以叫做程序。
- 代码： 可以被Java识别的内容叫做Java代码，可以被Python识别的内容叫做Python代码，其他编程语言也是一样。
- 编程： 使用编程语言编写代码的过程叫做编程。
