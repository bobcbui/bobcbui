## Java 简介

Java 是一种广泛使用的面向对象编程语言，具有跨平台、高性能、安全性强等特点，适用于 Web、移动、企业级应用开发。

## 开发环境搭建

1. 下载并安装 [JDK](https://www.oracle.com/java/technologies/downloads/)。
2. 配置环境变量 `JAVA_HOME`。
3. 安装常用 IDE，如 IntelliJ IDEA 或 Eclipse。

- 数据类型

## 面向对象编程

- 封装、继承、多态

- 集合框架（List、Map、Set）
- IO 流
- 异常处理
